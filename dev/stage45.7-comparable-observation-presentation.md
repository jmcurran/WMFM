# Stage 45.7: Comparable-observation presentation

Stage 45.7 adds a dedicated deterministic presentation panel for comparable-observation follow-up questions.

The panel:

- shows the resolved predictor settings used for matching;
- summarises the observed response across all selected neighbours;
- presents a responsive table containing the first five verified neighbours;
- reports source rows and similarity distances;
- states that similarity is limited to fitted-model predictors; and
- avoids presenting nearest-neighbour matching as proof of a bargain, unusual value, or causal difference.

The Stage 45.5 matching computation and Stage 45.6 deterministic answer remain unchanged. Unsupported requests display a bounded warning rather than an invented comparison.
