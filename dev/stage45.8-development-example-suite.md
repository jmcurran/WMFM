# Stage 45.8: Development example suite

Stage 45.8 adds a developer-only example suite for exercising the complete Stage 45 question boundary through the app.

The suite includes:

- lower-tail residual ranking for observations below fitted values;
- upper-tail residual ranking for observations above fitted values;
- two-sided ranking by absolute raw residual;
- comparable-case matching for a specified diamond;
- explicit conditional-percentile deferral; and
- deterministic rejection of residual inspection for a Poisson model.

All examples are marked as `developer` in the example metadata. They appear only when developer examples are enabled and remain hidden from the classroom example list.

The examples use existing packaged datasets and ordinary example specifications. They do not alter Stage 45 classification, computation, deterministic answers, or presentation behaviour.
