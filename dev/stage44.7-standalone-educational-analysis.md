# Stage 44.7: standalone educational analysis

## Purpose

Stage 44.7 makes the downloaded reproducible analysis independent of WMFM.
The document remains generated by WMFM, but its executable analysis code uses
ordinary R, the package supplying the selected data set, and `ggplot2` for the
model plot.

## Decisions

- The generated document must not load or call WMFM.
- The fitted equation is assembled explicitly from `coef(modelFit)`.
- Coefficient confidence intervals are calculated explicitly from
  `coef(modelFit)` and `vcov(modelFit)`.
- Linear-model intervals use a t critical value and the residual degrees of
  freedom.
- Generalised-linear-model intervals use a normal critical value.
- Logistic and Poisson coefficient intervals are also exponentiated to show
  odds ratios or expected-count multipliers.
- The model plot is expressed as ordinary `ggplot2` code using values recovered
  from the fitted model.
- Diagnostic plots continue to use standard model methods supplied by R.

## Scope limits

This stage makes the core generated document standalone. Dynamic contrast code
remains accepted only when it is already supplied as standalone R code. More
specialised predictor-effect plots and additional pedagogical annotation can be
added incrementally after the core standalone boundary is stable.
